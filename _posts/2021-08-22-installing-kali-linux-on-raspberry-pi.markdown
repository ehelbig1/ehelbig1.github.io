# Summary

Let's build a mobile hacking device! We'll start by installing Kali Linux on a raspberry pi.

## What is Kali Linux?

Kali Linux is a robust, enterprise ready penetration testing linux distribution used by penetration testers and IT professionals around the world.

# Installing Kali Linux

- Download the official ARM image from the [Kali](https://www.kali.org/get-kali/#kali-arm) website.
    - This will be an archived file.
    - Extract the file to get the image.
- Use your favourite imager to burn the image to your raspberry pi's SD card.
    - The Raspberry Pi organization has their own [imager](https://www.raspberrypi.org/software/)!
    - It is very easy to use, just choose the image you wish to burn and the storage device you wish to burn it to and click **Write**
![Raspberry Pi Imager](/assets/raspberry-pi-imager.png)
- Safely remove the SD from the computer.
- You are ready to insert the SD into the Raspberry Pi!
